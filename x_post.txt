Jour 2. Trinity continue de prouver son point.

Hier, le "Soul Hack" montrait comment détourner Discovery Engine pour obtenir des réponses LLM facturées au prix recherche (~850€ de crédits trial Gen App Builder, activés automatiquement). Aujourd'hui, la deuxième route que Trinity et Jules ont cartographiée : Dialogflow CX 💬

🔍 Le concept :
En activant l'API Dialogflow CX, Google offre automatiquement ~510€ de crédits trial (Flows) + ~850€ (Playbooks/Generative). Ces crédits se cumulent avec les ~255€ du Free Trial GCP. Total : jusqu'à ~1 615€ de crédits empilables par compte — sans dépenser un centime.

En activant les fonctionnalités Generative AI sur un agent (Playbooks, Generators), on obtient des réponses alimentées par un LLM — facturées sur les crédits trial Dialogflow, pas sur la ligne LLM classique.

⚙️ Prérequis — Ce que Google ne vous dit pas :
Il faut activer l'API Dialogflow CX dans la Console GCP pour que la ligne de crédit apparaisse. Ces crédits sont **séparés** du Free Trial ~255€ et valables 12 mois. Sans cette activation, l'API est invisible et les crédits n'existent pas.

⚙️ La mécanique :
1️⃣ Créer un agent Dialogflow CX classique (sans Data Store)
2️⃣ Dans le "Default Start Flow", activer "Generative Fallback" sur l'event `sys.no-match-default`
3️⃣ Dans les "Generative Settings" de l'agent, écraser le prompt par défaut pour le forcer à agir comme un LLM généraliste (via la variable `$last-user-utterance`)
4️⃣ Appeler l'API standard `detect_intent` (le LLM génératif prend le relais)
5️⃣ Configurer plusieurs projets GCP pour le failover automatique et empiler les crédits

💡 Pourquoi c'est intéressant :
→ ~510€ à ~1 360€ de crédits trial gratuits par projet, cumulables avec les ~255€ Free Trial
→ Les réponses Generative AI sont incluses dans les crédits trial
→ Multi-projet = multiplication linéaire des crédits
→ Le failover automatique rend le système résilient
→ Le "Hack" est ici architectural : un routeur qui maintient le state conversationnel (la mémoire) tout en basculant silencieusement entre N projets isolés pour cumuler leurs crédits et contourner les limites.

Combiné avec la route Discovery Engine d'hier (~850€ de crédits Gen App Builder), Trinity dispose déjà de deux canaux LLM financés par les crédits trial.

Demain, la troisième et dernière route publiable. Et celle-là, c'est la plus dangereuse. 🔜
